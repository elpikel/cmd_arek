Generate PDFs in Elixir

When you serch for pdf generator for elixir you most likely are going to find solution that is based on wkhtmltopdf which is not the best idea. There is great blog post on why using this library is considered harmful. Following advice from it we are going to use weasyprint which is BSD-licensed Python library that is almost as good as the commercial ones.

We have few possible ways to integrate with python library:

  • Make a system call
  • Expose python library as separate service and communicate with it using http
  • Use erlang ports
We are going to explore how we can use erlang ports to communicate with python library.
